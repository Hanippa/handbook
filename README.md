# מדריכי החברה

סביבת העבודה הזאת נבנתה עבור תוכן שייכנס ל-Guru בפורמט Markdown.

המטרה היא ליצור מדריכים קצרים, ברורים, וידידותיים לשימוש בזמן אמת.

## להתחלה מהירה

- [עמוד בית Guru טכני](./docs/guru/README.md)
- [בחירת תהליך לפי משימה](./docs/guru/start-here.md)
- [תבנית Guru לתהליך חדש](./docs/guru/templates/process-template.md)

## עקרונות התוכן

- כל דף מייצג משימה אחת בלבד
- כל שלב אומר בדיוק מה לעשות
- בכל מקום חשוב יש סימון ברור לתמונה
- כל מדריך מפנה למדריכים קשורים
- הניווט מתחיל מהמשימה של המשתמש ולא רק מהקטגוריה

## מה יש כרגע

- פתיחת קריאת תיקון
- מעבר למכשיר זמני
- קליטת מלאי חדש
- עמודי ניווט וקישורי המשך
