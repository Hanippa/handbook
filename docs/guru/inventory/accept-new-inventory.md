# קליטת מלאי חדש

## מתי משתמשים במדריך הזה

- **מטרה:** לקלוט ציוד חדש שמגיע לחברה ולרשום אותו נכון.
- **למי זה מיועד:** תפעול IT, מחסן, או אחראי מלאי.
- **מתי להשתמש:** כאשר מגיע משלוח חדש מספק או מהעברה פנימית.
- **זמן משוער:** 10 עד 20 דקות למשלוח.

## לפני שמתחילים

- [ ] להכין הזמנת רכש או תעודת משלוח
- [ ] לפתוח את מערכת המלאי
- [ ] לוודא אזור קבלה מסודר
- [ ] לבדוק אם נדרש אחסון מאובטח או טיפול מיידי

## מה צריך להכין

- שם הספק
- תאריך הגעה
- מספר הזמנה או משלוח
- דגמים
- כמויות
- מספרים סידוריים או מספרי נכס

## שלבים

### שלב 1: בדיקת המשלוח

לבדוק את מצב הקופסאות לפני פתיחה ולתעד נזק אם קיים.

**צילום מסך נדרש כאן:** דוגמה לנזק חיצוני או צילום של המשלוח כפי שהתקבל

![יש להוסיף כאן צילום מסך](../assets/images/accept-new-inventory/shipment-inspection.png)

### שלב 2: השוואה להזמנה

לוודא שהדגמים, הכמויות והאביזרים תואמים למסמכי ההזמנה.

**צילום מסך נדרש כאן:** תעודת משלוח או מסך הזמנה במערכת

![יש להוסיף כאן צילום מסך](../assets/images/accept-new-inventory/delivery-note-check.png)

### שלב 3: הזנת הפריטים למערכת

להוסיף כל פריט למערכת המלאי עם המזהים הנכונים.

**צילום מסך נדרש כאן:** מסך הזנת ציוד או שדות מספר סידורי / נכס

![יש להוסיף כאן צילום מסך](../assets/images/accept-new-inventory/inventory-entry.png)

### שלב 4: סימון סטטוס ותוויות

לסמן אם הפריט במלאי, שמור, או מוכן להגדרה, ולהוסיף תווית אם נדרש.

**צילום מסך נדרש כאן:** מסך סטטוס פריט או דוגמת תווית

![יש להוסיף כאן צילום מסך](../assets/images/accept-new-inventory/labeling.png)

### שלב 5: אחסון או המשך תהליך

להעביר את הפריטים לאחסון או לשלב הבא בתהליך.

**צילום מסך נדרש כאן:** מיקום אחסון, סטטוס סופי, או שיוך למחסן

![יש להוסיף כאן צילום מסך](../assets/images/accept-new-inventory/storage.png)
